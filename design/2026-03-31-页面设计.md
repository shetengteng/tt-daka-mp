## 1. 设计原则

- **极简**：每个页面只做一件事
- **高对比度**：黑白为主，彩色点缀区分打卡项目
- **触控友好**：打卡按钮大面积可点击
- **即时反馈**：打卡操作有动画和震动反馈

## 5. 日历页

**路径**: `pages/calendar/index`

**技术方案**：使用 `sar-calendar` 组件 + `formatter` 自定义日期样式（参考 tt-paikebao-mp 的 IncomeCalendar 实现）。

```
┌──────────────────────────────────┐
│  打卡日历                          │
│                                    │
│  ┌──────────────────────────────┐ │
│  │         sar-calendar           │ │
│  │  通过 formatter 自定义：       │ │
│  │  - 已打卡日: 绿色背景圆角      │ │
│  │  - 部分完成: 橙色背景圆角      │ │
│  │  - bottom 显示 "2/3" 完成数   │ │
│  │  - 未来日期: disabled          │ │
│  └──────────────────────────────┘ │
│                                    │
│  ── 3月31日 周一 ──                │
│                                    │
│  ┌──────────────────────────────┐ │
│  │ ┃ [图标] 运动    09:15 打卡   │ │
│  │ ┃ [图标] 阅读    21:30 打卡   │ │
│  │   [图标] 早起    未打卡        │ │
│  └──────────────────────────────┘ │
│                                    │
│  本月: 25/31 天  完成率 81%        │
│                                    │
│  [打卡]   [日历]   [统计]   [我的] │
└──────────────────────────────────┘
```

**sar-calendar formatter 实现**：

```javascript
const formatter = (day) => {
  const dateStr = formatDate(day.date)
  const dayData = calendarData[dateStr]
  if (!dayData) return

  const { done, total } = dayData
  if (done === total) {
    day.style = { backgroundColor: 'rgba(34,197,94,0.25)', borderRadius: '8rpx' }
    day.bottom = `${done}/${total}`
  } else if (done > 0) {
    day.style = { backgroundColor: 'rgba(249,115,22,0.25)', borderRadius: '8rpx' }
    day.bottom = `${done}/${total}`
  }
}
```

**交互**：

| 操作 | 行为 |
|------|------|
| 选中日期 | 下方展示该日打卡详情列表 |
| 切换月份 | sar-calendar 内置支持 |
| 点击详情记录 | 跳转项目详情 |

## 6. 统计页

**路径**: `pages/stats/index`

```
┌──────────────────────────────────┐
│  打卡统计                          │
│                                    │
│  ┌────────┐ ┌────────┐ ┌────────┐│
│  │   45   │ │   22   │ │   15   ││
│  │ 总打卡  │ │最长连续 │ │当前连续 ││
│  └────────┘ └────────┘ └────────┘│
│                                    │
│  ── 本周 ──                        │
│  一100% 二100% 三100% 四67% 五33% │
│  六--   日--                       │
│                                    │
│  ── 各项目 ──                      │
│  ┌──────────────────────────────┐ │
│  │ [图标] 运动  完成率92%        │ │
│  │ 连续15天 · 最长22天           │ │
│  │ 本月 28/31  最近7天 ●●●●●●○  │ │
│  └──────────────────────────────┘ │
│  ┌──────────────────────────────┐ │
│  │ [图标] 阅读  完成率85%        │ │
│  │ 连续7天 · 最长12天            │ │
│  │ 本月 25/31  最近7天 ●●○●●●●  │ │
│  └──────────────────────────────┘ │
│                                    │
│  [打卡]   [日历]   [统计]   [我的] │
└──────────────────────────────────┘
```

**计算规则**：

| 指标 | 说明 |
|------|------|
| 总打卡 | 至少完成 1 个项目的天数（去重） |
| 当前/最长连续 | 从今天往前连续有打卡的天数（跳过非目标日） |
| 本周完成率 | 每天 = 已打卡项目数 / 应打卡项目数 |
| 最近7天 | ● 已打卡 ○ 未打卡 -- 非目标日 |
